# vue项目适配显示markdown

## 必须依赖

```bash
npm install marked front-matter highlight.js
```

## 实现方案

### 1. 基础配置
```javascript
// markdown.js (工具文件)
import { marked } from 'marked'
import hljs from 'highlight.js'

// Markdown 配置
marked.setOptions({
  breaks: true,
  gfm: true,
  highlight: (code, lang) => {
    return hljs.highlightAuto(code, [lang]).value
  }
})

// Front Matter 解析
export const parseMarkdown = (raw) => {
  const { attributes, body } = fm(raw)
  return {
    meta: attributes,
    html: marked.parse(body)
  }
}
```

### 2. 组件化方案
```javascript
<template>
  <article class="markdown-body" v-html="processedHtml"></article>
</template>

<script setup>
import { computed } from 'vue'
import { parseMarkdown } from '@/utils/markdown'

const props = defineProps({
  content: String,
  inline: { type: Boolean, default: false }
})

const processedHtml = computed(() => {
  const { html } = parseMarkdown(props.content)
  return props.inline ? html.replace(/^<p>|<\/p>$/g, '') : html
})
</script>

```

## 其他要点

### 1. 安全处理
```javascript
// 使用 DOMPurify 防止 XSS
import DOMPurify from 'dompurify'

const safeHtml = computed(() => {
  return DOMPurify.sanitize(processedHtml.value)
})
```

### 2. 动态加载方案
```javascript
// 动态加载.md文件
const loadArticle = async (id) => {
  try {
    const content = await import(`@/posts/${id}.md?raw`)
    return parseMarkdown(content.default)
  } catch (e) {
    return {
      meta: { title: '加载失败' },
      html: '<p>文章加载失败</p>'
    }
  }
}
```

### 3. 目录生成
```javascript
const generateTOC = (html) => {
  const div = document.createElement('div')
  div.innerHTML = html
  return [...div.querySelectorAll('h2,h3')]
    .map(heading => ({
      level: parseInt(heading.tagName.substring(1)),
      text: heading.textContent,
      anchor: heading.id
    }))
}
```
